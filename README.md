# PetriSeed

PetriSeed is a data-driven artificial life sandbox built in Godot 4.6 C#.

Starting from a handful of simple microbes, players observe and influence the emergence of complex ecosystems through mutation, adaptation, predation, environmental pressure, and evolutionary change.

## Core Pillars

- Artificial Life
- Emergent Gameplay
- Evolution & Mutation
- Ecosystem Simulation
- Procedural Generation
- Scientific Inspiration

## Current Status

Pre-Alpha — Foundation Architecture

## Technology

- Godot 4.6
- C#
- Data-Driven Simulation
- Deterministic Simulation Ticks
- JSON Configuration Pipeline
